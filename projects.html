Create a Ruby on Rails web application where user's can securely create, store & manage their personal notes (memos) dynamically from a single page by utilizing Ajax and jQuery on the client side.

OBJECTIVE

Create tetris game using pure JavaScript and HTML5 canvas. Each new random piece/shape is created one at a time and travel down the board in intervals of 1 second. As shape is drops, player can rotate or move shape, pause the game, drop shape (overriding the 1 second interval). If a row on the board is completely filled, row is cleared & removed from the board and 10+ points are added to the player's score.

FEATURES

  • ✓ Game start and pause button controls to begin game or pause in the middle of a game.
  • ✓ Randomized shape generation.
  • ✓ Shape travels down board at 1 second intervals.
  • ✓ Player can rotate and move shapes left, right & down (expediting regular drop interval).
  • ✓ Shape pieces are stack on top of each other until limit is reached and game is lost.
  • ✓ Each time entire row on board is filled by shapes, it is cleared and removed from the board (expanding limit threshold).
  • ✓ Score is incremented by +10 points each time row is cleared.
  • ✓ Additional numpad key shortcuts provided.
